Projects

Bunny Hop

A simple 2D platformer game. Use the left and right arrow keys to move the bunny and the up arrow key to make the bunny jump. The Game Assets were made by Kenney Vleugels (www.kenney.nl) and were found on opengameart.org. This project was created using HTML, Javascript, and Phaser.

Tower of Hanoi

The classic Tower of Hanoi game. The objective is to move all of the discs from the left tower to the right tower. A larger disc cannot be placed on top of a smaller disc. This project was created using HTML, CSS, and Javascript.

Home Baked Recipes

A recipe tracker where users can upload their recipes and view uploads from other users. This project was created using Ruby on Rails and CSS and deployed on Heroku.
